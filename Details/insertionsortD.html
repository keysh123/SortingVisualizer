    • Insertion Sort

    Insertion sort is a simple and efficient sorting algorithm that builds the final sorted list one item at a time. It works by repeatedly taking an element from the unsorted part of the list and placing it in its correct position within the sorted part of the list. Here are the details of how insertion sort works:

    1.Initialization : Start with the first element in the list, considering it as the sorted sublist, and the remaining elements as the unsorted sublist.
    2.Iterative Comparison and Insertion : For each element in the unsorted sublist, compare it with the elements in the sorted sublist. Starting from the rightmost element in the sorted sublist, move elements to the right until you find the correct position for the element from the unsorted sublist. This process effectively inserts the element into its correct position within the sorted sublist.
    3.Expanding the Sorted Sublist : After processing all elements in the unsorted sublist, the sorted sublist has increased in size by one, and the unsorted sublist has decreased in size by one.
    4.Repeat : Continue this process, taking one element at a time from the unsorted sublist, and inserting it into its correct position in the sorted sublist. Repeat until the entire list is sorted.
    5.Termination : The algorithm terminates when the unsorted sublist becomes empty, and the sorted sublist contains all the elements in the sorted order.

    Insertion sort has the following characteristics :

    • Time Complexity : In the average and worst cases, insertion sort has a time complexity of O(n^2), where n is the number of elements in the list. However, it is more efficient than bubble sort and selection sort for small datasets and for lists that are nearly sorted.
    • Stability : Insertion sort is a stable sorting algorithm, meaning it maintains the relative order of equal elements.
    • In-Place : Insertion sort is an in-place sorting algorithm, meaning it doesn't require additional memory for temporary storage.
    • Adaptive : The performance of insertion sort can be adaptive, meaning it can be faster when dealing with partially sorted lists, as it requires fewer comparisons and moves in such cases.

    Insertion sort is a practical sorting algorithm for small lists or when dealing with lists that are nearly sorted. It is often used in combination with more advanced sorting algorithms, like quicksort or mergesort, in hybrid sorting techniques. While it's not the most efficient algorithm for very large datasets, its simplicity and low overhead make it a valuable sorting method in many situations.

    • Bubble Sort

    Bubble sort is a straightforward and inefficient sorting algorithm that works by repeatedly stepping through a list of elements and comparing adjacent pairs, swapping them if they are in the wrong order. The process continues until the entire list is sorted. Here are the details of how bubble sort works:

    1.Initialization : Start at the beginning of the list.
    2.Comparison and Swap : Compare the first two elements in the list. If the first element is greater than the second, swap them. If they are in the correct order, leave them as they are. Continue this process for each pair of adjacent elements in the list, moving from left to right.
    3.One Pass Through the List : After the first pass through the list, the largest element will have "bubbled up" to the end of the list. The largest element is guaranteed to be in its correct position.
    4.Repeat for Unsorted Elements : Repeat the comparison and swap process for the remaining unsorted elements in the list, which is all the elements except the last one. This means you will ignore the last element in each pass since it is already in its correct position.
    5.Continue Until Sorted : Continue these passes through the list until no swaps are made during a pass. When no swaps are needed, it indicates that the list is fully sorted.
    6.Termination : The algorithm terminates once the entire list is sorted, and you no longer need to make any swaps.

    Bubble sort is a very simple sorting algorithm, but it has some important characteristics and limitations:

    • Time Complexity : In the worst and average cases, bubble sort has a time complexity of O(n^2), where n is the number of elements in the list. This makes it inefficient for sorting large lists.
    • Stability : Bubble sort is a stable sorting algorithm, meaning it maintains the relative order of equal elements.
    • In-Place : Bubble sort sorts the list in-place, meaning it doesn't require additional memory for temporary storage.
    • Adaptive : Bubble sort is adaptive, meaning its performance can be improved if the list is partially sorted. In such cases, it may have a lower time complexity.
    • Simple Implementation : Bubble sort is easy to understand and implement, which makes it useful for educational purposes and for small datasets.

    Bubble sort is not a practical choice for sorting large datasets due to its poor time complexity. More efficient sorting algorithms like quicksort, mergesort, or heapsort are generally preferred for real-world applications.
